## Installation:
All builds and downloads are available [here](https://github.com/meadowcottage/streamlabs/releases) in the repo releases section.

## Development

Installation:
```
$ git clone https://github.com/meadowcottage/streamlabs.git
$ cd streamlabs
```

Running:
```
$ npm start
```

Packaging:
```
$ npm run package
```
